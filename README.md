# Python-Mini-Projects 🚀

A collection of 4 Python CLI tools built in 1 day by Priyadharshini!

### 📋 Projects List:

1. **Student-Grade-Calculator** - `student.py`
   - Takes student name & marks, calculates grade with if-elif-else
   
2. **House-Price-Prediction** - `house_price.py`
   - Predicts if house is `Luxury House`, `Modern House` or `Affordable House`
   
3. **Iris-Flower-Classifier** - `Iris.py`
   - Classifies flowers: `Iris Setosa`, `Iris Versicolor`, `Iris virginica`
   
4. **Spam-Detector** - `spam.py`
   - Detects spam messages/calls. Output: `Prediction:Safe` or `Prediction:Possible Spam`
